#### 입력이 100 이하인 경우
* 완전 탐색
* 백트래킹
#### 입력이 10,000 이하인 경우
* 최대 O(n^2^) 이내로 끝내야하는 문제
* 경우에 따라 O(n^2^ log n) 까지는 허용
#### 인력이 1,000,000 이하인 경우
* 최대 O(n log n)으로 끝내야하는 문제
* 힙
* 우선순위 큐
* 정렬
* 동적 계획법
* 위상 정렬
* 다익스트라 알고리즘
#### 인력이 100,000,000 이하인 경우
* 최대 O(n)으로 끝내야하는 문제
* 동적 계획법
* 그리디
#### 그 이상
* 최대 O(log n)으로 끝내야하는 문제
* 이진 탐색
#### 실시간 정렬이 이루어져야 하는 경우
* 우선순위 큐 혹은 힙 사용
